Support Vector Machines

Support Vector Machine (SVM) is a supervised learning model used for both regression and classification tasks; they are more commonly used for classification problems. On a high level, it operates by finding the optimal hyperplane to separate classes and maximize the distance between them to minimize error.

Mathematical Basis

Key Terms:

  • Hyperplane: the decision boundary separating different classes in the feature space.
  • Support Vectors: the closest data points to the hyperplane.
  • Margin: distance between hyperplane and support vectors.
  • Kernel: function that maps data into a higher dimensional space to handle data that is not linearly separable.
  • C: regularization term that balances margin maximization and misclassification penalties. A higher C forces a stricter penalty on errors. In the following image, L2 would be chosen as the best hyperplane because it maximizes the distance between the hyperplane and the nearest points.


If there is an outlier, the SVM can ignore it when finding the best hyperplane. When using a hard margin, the SVM may include more outliers to create the maximum distance between classes; this could lead to overfitting. When using a soft margin, the SVM will ignore outliers, allowing for some misclassifications to improve generalizability.

Kernels:

Data is considered to be not linearly separable if a straight line cannot divide classes. Kernels are used to map data that is not linearly separable onto a higher dimension where it can be separated by a straight line. There are different types of kernel functions that can be used, including a linear kernel, polynomial kernel, or radial basis function (RBF) kernel.

Implementation

This is an example of using an SVM for classification. Feel free to follow along on Colab or another coding platform!

  1. The first step is to import the necessary packages.
    • Load_breast_cancer is a function that allows us to access a data set that is originally from the Diagnostic Wisconsin Breast Cancer Database. The dataset includes 30 different features describing characteristics of a breast mass for 569 different samples. 2 Each sample is classified as malignant or benign. The goal of the SVM is to classify samples as malignant or benign.
    • Matplotlib.pyplot will help visualize the results in a graph.
    • DecisionBoundaryDisplay will display the decision boundary produced by the SVM in a graph.
    • Train_test_split will split the input data into training and test sets.
    • SVC is a Support Vector Classifier from Scikit-learn
  2. Then, we will load the breast cancer dataset into an X array containing the features for each sample and a y array containing the target classification (malignant or benign) for each sample. For this example, we are only using two of the explanatory features so the hyperplane can be visualized in a graph. However, feel free to run the SVM using all of the features and compare the results to using only two features.

    We will also split the data into training and testing sets with a test size of 20% using the train_test_split function.
  3. Next we will create an SVM object. We set the kernel to linear which assumes our data is linearly separable. We also set the C value equal to 1 which is a standard value for the regularization parameter. There are many other hyperparameters that can be set and you can explore them on the SVC documentation.

    We will also fit our data to the training data and make predictions on the test data, storing them as y_pred.
  4. Finally, we will display the results of the SVM using the Decision Boundary Display function. Specifically, we use the DecisionBoundaryDisplay.from_estimator function to plot the decision boundary of the svm object we created. We input the estimator (our SVC) and input dataset X. The function will plot all of the points in the input dataset and color them according to the output of the SVM. We will use matplotlib pyplot (plt) to display the scatterplot. Useful Applications

SVM is commonly used for classification problems and occasionally for regression problems. It works well with high dimensional data, meaning there is a large number of features, that can be problematic for other models. Some common applications are:

  • Face detection: SVMs classify parts of an image as face and non-face.
  • Text categorization: classify documents into different categories such as news, emails, or webpages.
  • Image classification: identify objects within images.
  • Bioinformatics: classify people based on genetic sequences or other biological data.3
