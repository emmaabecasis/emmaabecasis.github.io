Application Projects

Machine learning is a powerful tool used in nearly every area of work. This page provides ideas of projects to apply machine learning related to business, healthcare, and sports/gaming. Machine learning can be used in countless other ways than the ideas given here, but these are a great place to get started using machine learning! These projects can be carried out using open-source data on any coding platform. Google Colab is a free online coding platform for anyone with a Google account.

The open source data sets described here are housed in the University of California Irvine Machine Learning Repository. This repository contains 689 datasets that are useful for machine learning tasks. If you follow the links to the dataset, you can use the “Import in Python” button on the UC Irvine site to find the exact code needed to download the dataset.

Category Name Dataset Task
Business Bank Marketing This dataset contains 41,118 samples with 20 explanatory variables each. The input variables contain information about clients and phone calls of a Portuguese banking institution. The target variable is whether or not the client will subscribe to a bank term deposit. Use a classification model to predict if a certain client will submit a deposit.
Online Retail This dataset contains information on 12,330 transactions on an online retail site during a one-year period. There are 17 input features about each user’s visit to the site. The target variable is whether or not a user made a purchase. Use a classification model to predict if a certain user will make a purchase.
Real Estate Valuation This dataset contains information on 414 houses in New Taipei City, Taiwan. There are six inputs related to each house including the house age and distance to the nearest metro station. The output is the house price of unit area. Use a regression model to predict the price of a house based on its features.
Healthcare Estimating Obesity Levels This dataset contains 2,111 samples with information on 16 input features. The inputs include health, physical activity, and family information. The output is obesity level, ranging from Insufficient Weight to Obesity Type III. Use a classification model to predict the level of obesity based on eating habits and physical condition.
Heart Disease This dataset contains 313 samples with 13 different input features related to health information. The output variable is heart disease diagnosis; it ranges from 0 to 4, with 0 being no diagnosis. Change all values that are 1, 2, 3, or 4 in the “num” column (representing diagnosis of heart disease) to be 1. This will make 1 represent diagnosis of heart disease while 0 represents no diagnosis. Then, use a classification model to predict heart disease diagnosis.
Parkinson's Disease This dataset includes information on 31 people of which 23 are diagnosed with Parkinson’s Disease. Each row indicates a voice recording from one of these individuals and each column is a specific voice measure. The output variable is in the “target” column. 1 indicates Parkinson’s Disease and 0 indicates no Parkinson’s Disease. Use a classification model to predict if someone has Parkinson’s disease based on voice information.
Sports and Gaming Poker This dataset contains 1,025,010 data points with each data point representing a hand of five playing cards from a deck of 52. There are ten input features, including suit and rank information for all five cards. The output variable is the poker hand. The variable is integer-valued from 0 to 9. Information on what each integer indicates can be found on the UCI website. Use a multiclass classification model to predict poker hands.
Rocket League Skillshots This dataset contains 297 datapoints. There are 18 input features including player information and in-game metrics. The target variable (“class_number”) is the skill a user is performing, such as ceiling shot or front flick. Use a multiclass classification model to predict activity type in Rocket League.
